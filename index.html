Contribution and Reflection in 2A Design Inspiration

My Contributes

In this period, I was asked to find at least one dataset in data.gov.au, which must have API access option to allow us request data By Ajax to make an interactive website. I found some datasets about parks in Brisbane. The left one is the location of those, and the right one is the datasets contains facilities in each park. By selecting position in the map or filtering items, users could search park in Brisbane with some specific conditions.

Reflection

My poster is quite success since I got some positive feedback that my project is full of interactions and interests. Through this stage of study, I learned some excellent knowledge such as how to make a low-fidelity prototype and story board, how to use technique of Adobe Illustrator in Creative Cloud. However, there are some points need to improve. For example, my poster is quite like a function, not a project. To make it more completable, I will find some other classmates during the contact class to be a team. And we will brainstorm some nice ideas to implement a interesting and interactive website.

2D Work-in-Progress

Contribution and Reflection in 2D Prototype

Implement Home Page

Before the presentation of Work-in-Progress, I implement the HTML, CSS code of home page, this is the initial page when user click on our project. Also, I add a hover animation if mouse hover on it to hint users click it.

Implement Player Selection Page

I also implement the HTML and CSS of player selection page to allow users select the number of players. Each image has animation if mouse hovered on to prompt user click.

Implement Chess Board Page

Also, I implement the HTML, CSS of chess board page, which is the most significant part of project. At the top of the page is the player information box more speciic introduce are as follows:

Dynamic Initialize Player Info Box

I implement the algorithm to dynamically initialize player information box at the top of chess board page. By achieve so, I pass the information into this page from Player Selection page and packaged a method to get the number of players. Then use JQuery to fill in text into player information box.

Animation of Chess Moving

In order to vivid representing the movement of chess, I packaged some methods to implement animations. To doing so, I first achieve to make given ID of chess board flashing. And then to make a set of boards flashing one by one, just use setTimeOut function in JQuery and accumulate the time of delay.

Main Game Logic

Response to actions when chess move to specific building. Some pieces of code are shown above

If player move to a community hall at first time, he/she will own this community hall and get number of stars.

If player move to a community hall which is belong to other player, he/she will pay rents to that player.

If player move to a community hall which is belong to himself/herself, nothing happened.

If player move to a park, there are four parks, and four special rewords, or penalties will occur.

API

I successfully request data from online database by Ajax learned in workshop.

Reflection

I believe I perfectly finished the code part in the stage of Work-in-Progress. During the presentation, we got many positive feedback about the implement of website. However, we noticed a big problem is that I just use data of community halls and parks in Brisbane. And most data are community halls, which is conflict with our purpose. We were expected to build an interactive website to introduce Brisbane, however, too much community halls is not attractive enough for our target users. To solving this problem, we made some emergency discussion during mid break and fixed it before final demonstration.
